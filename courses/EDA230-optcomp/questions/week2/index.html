Insertion of PHI functions

PHI function for x is inserted in union of DF(v) where v is a vertex where x is assigned.

Using the iterative approach, if we have assignments in node 0, 2 and 7 we would take the union of their dominator frontiers (in this case) 3, 4, and 6 resulting in:

DF1(A) = { 3, 4, 6 }
A = { 0, 2, 7 }

For each of these we will take the union of their respective DF. I.e.

DF2(A + DF1(A)) =  DF(0) + DF(2) + DF(7) + DF(DF1(A))

Then iterate untill the set is unchanged.
